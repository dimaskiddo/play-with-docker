Play with Docker (PWD) is a labs created by Marcos Liljedhal and Jonathan Leibiusky.

Play with Docker is a playground which allows users to run Docker commands in a matter of seconds. It gives the experience of having a free Virtual Machine in browser, where you can build and run Docker containers and even create clusters in Docker Swarm Mode. Under the hood Docker-in-Docker (DinD) is used to give the effect of multiple VMs/PCs.
